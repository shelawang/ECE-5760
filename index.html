FPGA Command Line Interface and File System

ECE 5760 Final Project, Spring '15

Victor Fei (vf63), Shela Wang (sw679), and Richard Quan (rq32)

Introduction

Our project connects a VGA monitor and a keyboard to an FPGA on an Altera DE2 development board to increase the usability of the system. By simulating a traditional command line interface on the screen, we have allowed users to perform some operations on the fly without having to compile and upload code every time. Additionally, we utilized the DE2 boardâ€™s inbuilt SD card reader to allow users to import, manipulate, and save portable data, effectively creating a file system for the FPGA.

Block Diagram

The figure below illustrates the major components of the system. The hardware interface consists of an FPGA on the DE2 board with an SD card reader connected to a PS2 keyboard and a VGA monitor. We use a Pancake processor running on the FPGA to interface between the software (Syrup) and hardware (Verilog) components of our design. The FPGA accepts user input via the keyboard, interprets and processes the command through Pancake, and outputs the I/O through a command line interface (CLI) displayed on the VGA, in a way reminiscent of personal computers of yore. The SD card reader sends and receives it data in the same fashion.
